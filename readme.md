Существует текстовый файл со следующей структурой:
$MKPRG,CFG:R,1,GGA:11100,GNS:11100, ... ,OTHER:11000,B:38400,T:1
и таких 8 строк
где 1 - номер канала
GGA,GNS - message id
11100 - настройки отправки сообщений
B - baudrate
T - период
Необходимо написать скрипт на Python, который прочитает данный файл и сформирует массив словарей.
Для примера выше этот массив должен выглядеть следующим образом:
[
    {
        ChannelNumber: 1,
        B: 38400,
        T: 1
        GGA: "11100",
        GNS: "11100",
        ...
        OTHER: "11000"
    }
]
Также необходима возможность выполнить обратное преобразование: создавать текстовый файл заданного формата по массиву словарей.
